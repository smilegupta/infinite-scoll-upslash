## Image Gallery
### Features
- Exploring Images
- Searching Images
- Upload Image and Getting Sharable Link

### Tech Stack
- Frontend : Nextjs
- Styling: CSS and Bootstrap
- Deployment: Vercel

### API Consumed
- 1) Unsplash Image API for listing and searching
- 2) Self wriiten api to upload files to S3 bucket
